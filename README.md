# Hotel Booking Mini System (Dockerized)

Full-stack mini hotel booking system using:
- React + TypeScript (frontend)
- .NET 8 Web API (backend)
- PostgreSQL (database)
- Docker Compose (local orchestration)

## Planned Features (MVP)
- View hotels, rooms
- Create bookings with date validation + no-overlap rule
- Admin CRUD for hotels/rooms + bookings list

## Run 
`docker-compose up --build`
